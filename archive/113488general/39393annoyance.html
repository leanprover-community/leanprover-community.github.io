Recall that (∘) f g x = (f ∘ g) x = f (g x). So

   ((∘) ∘ (∘)) a b c d
=  (∘) ((∘) a) b c d
=  ((∘) a) (b c) d
=  (∘) a (b c) d
=  a (b c d)

Kevin Buzzard (May 17 2018 at 22:48):
