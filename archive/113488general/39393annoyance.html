Recall that (∘) f g x = (f ∘ g) x = f (g x). So

   ((∘) ∘ (∘)) a b c d
=  (∘) ((∘) a) b c d
=  ((∘) a) (b c) d
=  (∘) a (b c) d
=  a (b c d)

Kevin Buzzard (May 18 2018 at 00:48):
