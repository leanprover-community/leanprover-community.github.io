found myself in need of some small generic results while messing with some stuff in analysis/normed_spaces (#726).
figured they might be sort of useful in other contexts, so tossing in a separate PR.

  • for $1 < x$, $1 < y$ in a linear ordered ring, there exists natural number $n$ such that $y^n \le x < y ^ {n+1}$
  • for $0 < x$, $1 < y$ in a linear ordered field, there exists integer $n$ such that $y ^ n \le x < y ^ {n+1}$
