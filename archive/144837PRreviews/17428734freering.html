Mario Carneiro (May 10 2019 at 20:48):

It doesn't need to be noncomputable. It is coming from this theorem:

noncomputable def subsingleton_equiv [subsingleton α] : list α ≃ multiset α :=
{ to_fun := coe,
  inv_fun := quot.out,
  left_inv := λ l, list.ext_le
    (by { rw [← multiset.coe_card, ← multiset.coe_card], congr' 1, exact quot.out_eq _ }) $
    λ n h₁ h₂, subsingleton.elim _ _,
  right_inv := λ l, quot.out_eq _ }

This generalizes to the basic equiv theorem that quot A R ~= A when \all x y, R x y -> x = y. The inverse function is defined by quot.lift, not quot.out
