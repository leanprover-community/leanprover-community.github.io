Scott Morrison (Feb 28 2019 at 10:40):

As a step along the way to nat_cases, and I think independently useful, I've significantly upgraded the fin_cases tactic.

Scott Morrison (Feb 28 2019 at 10:41):

It used to just case bash on fin n hypotheses; now it can case bash finset, list, and fintype hypotheses.

Scott Morrison (Feb 28 2019 at 10:42):

e.g.

instance (n : ℕ) : decidable (prime n) := decidable_prime_1 n
example (x : ℕ) (h : x ∈ (range 10).filter prime) : x = 2 ∨ x = 3 ∨ x = 5 ∨ x = 7 :=
begin
  fin_cases h; exact dec_trivial
end
