Kevin Buzzard (Mar 13 2019 at 21:50):

def bit0 {α : Type u} [s  : has_add α] (a  : α)                 : α := a + a
def bit1 {α : Type u} [s₁ : has_one α] [s₂ : has_add α] (a : α) : α := (bit0 a) + 1

This is what Lean uses when it sees numeric characters.
