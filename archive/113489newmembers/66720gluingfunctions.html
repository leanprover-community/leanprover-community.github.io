Mario Carneiro (Jun 04 2019 at 22:27):

I would say this is not idiomatic, in that you should avoid talking about fn or get if you can avoid it, in favor of the mem relation. Here is a theorem you can prove about the function value without the equality constraint:

theorem glue_mem {f : ι → α →. β} {a : α}
  (ha : ∃ i, a ∈ dom (f i)) : ∃ i b, b ∈ f i a ∧ b ∈ glue f a :=
sorry

theorem glue_eq_mem {f : ι → α →. β}
  (H : ∀ i i' a b b', b ∈ f i a → b' ∈ f i' a → b = b')
  {i : ι} {a : α} {b : β} (ha : b ∈ f i a) : b ∈ glue f a :=
begin
  cases ha with h1 h2,
  use ⟨i, h1⟩,
  exact H _ i a _ b ⟨_, rfl⟩ ⟨h1, h2⟩,
end
