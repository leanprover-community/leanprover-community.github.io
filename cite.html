How do I cite mathlib? #

If you use mathlib in your work, please cite it using the following reference:

The mathlib Community. The Lean Mathematical Library. In Proceedings of the 9th ACM SIGPLAN International Conference on Certified Programs and Proofs (CPP '20), January 20â€“21, 2020, New Orleans, LA, USA. ACM, 2020. DOI: 10.1145/3372885.3373824

BibTeX #

{% raw %}

@inproceedings{mathlib2020,
  author = {{The mathlib Community}},
  title = {The {L}ean Mathematical Library},
  booktitle = {Proceedings of the 9th ACM SIGPLAN International
               Conference on Certified Programs and Proofs},
  series = {CPP '20},
  year = {2020},
  month = {January},
  location = {New Orleans, LA, USA},
  publisher = {ACM},
  doi = {10.1145/3372885.3373824},
  url = {https://doi.org/10.1145/3372885.3373824}
}

{% endraw %}

arXiv preprint #

The arXiv preprint is available at arxiv.org/abs/1910.09336.

GitHub citation #

The mathlib4 repository also contains a CITATION.md file, which GitHub recognizes. You can also click the "Cite this repository" link on the repository page to view the above BibTeX citation there.
